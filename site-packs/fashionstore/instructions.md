# Fashionstore operational notes

- Начинай со страницы `https://fashionstore.ru/`.
- Для поиска используй формы с `q` в шапке (`search-form`/`fixed-search-form`), сначала `Enter`, затем кнопку поиска.
- Для открытия карточки товара кликай только по видимым ссылкам из листинга (`.product`) с путями `/catalog/...`.
- Учитывай, что ассортимент и ссылки в женском/мужском контексте могут отличаться.
- На карточке товара перед `add_to_cart` проверь, что выбран доступный размер/вариант, если сайт этого требует.
- Для `add_to_cart` сначала используй явные селекторы CTA (`product_add_button`), затем fallback по текстам (`Добавить в корзину`, `В корзину`).
- На карточке могут быть дубли desktop/mobile кнопок, поэтому кликай только по видимым целям.
- После `add_to_cart` может открываться модалка `Товар в корзине`; если планируется добавить ещё товары, нажимай `Продолжить покупки`.
- В модалке `Товар в корзине` элемент `Продолжить покупки` является ссылкой, а не кнопкой: используй `#modal-window .close-click` или `role=link` с именем `Продолжить покупки`.
- Перед шагом `open_cart` убедись, что модалка добавления закрыта, иначе она может перехватывать клики по корзине в шапке.
- Считай добавление успешным только при изменении UI корзины или при переходе в `/personal/cart/`.
- Для `open_cart` сначала используй иконку корзины в шапке, затем fallback на ссылку `/personal/cart/`.
- Если в процессе появляется login-gate (`Войти`, телефон, код из СМС), фиксируй это как ручной шаг.
- Останавливайся перед финальным подтверждением заказа, оплатой и другими необратимыми шагами.
