# Filterix operational notes

- Начинай со страницы `https://filterix.ru/`.
- Перед действиями закрывай/подтверждай попап города, если он перекрывает интерфейс.
- Основной поиск используй через шапку: `form[action='/poisk/']` и `input#search_items`.
- Для поиска сначала пробуй `Enter` в поле, затем клик по иконке/кнопке поиска.
- Для открытия карточки товара используй только видимые ссылки из выдачи с путём `/product/...`.
- Не используй `title` страницы как основной критерий распознавания search/cart-состояний: на `/poisk/` он может быть пустым.
- Для `add_to_cart` сначала пробуй явные pack-селекторы (`button.add_item`), затем текстовые кнопки (`В корзину`).
- Считай добавление успешным только при изменении UI корзины: счётчик, состояние `В корзине`, переход на `/basket/` или другие корзинные сигналы.
- Для `open_cart` сначала кликай по `a.small_basket` (даже при `href="#"`), затем используй ссылки с `/basket`.
- Не используй `/cart` как fallback: этот маршрут на сайте возвращает 404.
- Если прямой переход на `/basket/` вернул на главную, повторно открывай корзину через UI-элемент в шапке.
- Если после перехода к оформлению появилась форма входа (`Войти`, `Электронная почта`, `Пароль`), фиксируй это как login-gate.
- Останавливайся перед финальным подтверждением заказа, оплатой и другими необратимыми шагами.
