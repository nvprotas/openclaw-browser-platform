# Gemotest operational notes

- Начинай со страницы `https://gemotest.ru/`.
- Перед действиями учитывай попап/подтверждение города, если он перекрывает элементы страницы.
- Для поиска используй основную форму `form[action='/search/']` и поле `input[name='q']`.
- Для отправки поиска сначала пробуй `Enter` в поле, затем кнопку `Найти`.
- Для открытия карточки выбирай только видимые ссылки анализов из выдачи (`a.analysis-item__title[href*='/catalog/']`).
- Для `add_to_cart` сначала используй явные видимые селекторы (`.research-cart-button.js-add-to-cart:visible`, `.js-add-to-cart:visible`), затем текстовый fallback `В корзину`.
- На сайте есть скрытые дубли CTA-кнопок, поэтому клики по неуточненным селекторам часто попадают в невидимые элементы.
- Считай `add_to_cart` успешным только если в `/cart/` виден товар в блоке корзины или появляются элементы оформления (`Выбрать отделение`, `Применить`).
- Для `open_cart` сначала кликай `a.basket.js-header-basket-container`, затем используй fallback `a[href='/cart/']`.
- Пустую корзину распознавай по сигналам `Корзина пуста` и `.empty_page`.
- Перед продолжением оформления может требоваться выбор отделения; фиксируй это как gate для ручного шага.
- Останавливайся перед финальным подтверждением заказа, оплатой и другими необратимыми шагами.
